Link Element

For some reason, I constantly have trouble remembering how to link my stylesheet to the index html file, so it seems only obvious that I should start with this one.

Tag Name Description
<link> Link Element It's used to link html documents to external resources like CSS files. It commonly uses the atributes bellow
href Hyperlink attribute Specifies the URL of the external resource
rel Relationship attribute Specifies the relationship of the linked document to the current document
type Type attribute Defines the type of content being linked
